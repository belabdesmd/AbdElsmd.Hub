# Shortcuts and Commands

| Action                    | Shortcut                |
| ------------------------- | ----------------------- |
| Create/Go To File         | `Ctrl + O`              |
| Switch Preview/Edit Modes | `Ctrl + E`              |
| Search Every File         | `Ctrl + Shift + F`      |
| Move Forward/Back         | `Ctrl + Alt + <- or ->` |
# Theming

- ==Vanilla Amoled== as main Theme.
- ==JetBrains Mono== as Interface and Text Fonts.
- `#8A8A8A` as Accent Color.
- ==CSS Snippets== for further customization, you can use with the property `cssclasses`.
	- ==Notebook Backgrounds== on `.obsidian/snippets/Notebook Background.css` to define main backgrounds: For now use only .page-manilla.
	- ==Pillar Note Themes== on `.obsidian/snippers/Note Themes.css` to define themed notes (backgrounds and colors), you need first to define class `.page`
		- `.green` for pages with <span style="color:rgb(146, 230, 73)">green theme</span>
		- `.red` for pages with <span style="color:rgb(226, 44, 60)">red theme</span>
		- `.blue` for pages with <span style="color:rgb(0, 32, 96)">blue theme</span>
		- `.orange` for pages with <span style="color:rgb(237, 96, 34)">orange theme</span>
		- `.tangerine` for pages with <span style="color:rgb(246, 88, 72)">tangerine theme</span>
		- `.yellow` for pages with <span style="color:rgb(255, 214, 61)">yellow theme</span>
